# ProyectoMultiagentesNovedadesOrdenPublico
El proyecto propone el desarrollo de un sistema de generación de reportes apoyado con inteligencia artificial generativa, orientado a eventos de orden público en el departamento del Cauca. Se concibe como una iniciativa de alcance reducido cuyo objetivo principal es evaluar la viabilidad y desempeño de técnicas de IA Generativa.
